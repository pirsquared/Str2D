General string parsing and manipulation functions

str2d.utils.apply_mask(inp: str = '', mask: str = '', char: str = ' ', substitute_char: str = None, invert: bool = False) → str[source]¶

Mask a string

Parameters:
  • inp – str, the string to apply the mask to
  • mask – str, the string used as mask on inp
  • char – Character in mask that does the masking - default is ` `
  • substitute_char – What to replace char with. If None # replace_with? uses characters from msk
  • invert – We could invert the mask
Returns:

str

 
>>> apply_mask(inp='abcdefghij', mask='-----     ', invert=True)
'abcde     '

>>> apply_mask(inp='abcdefghij', mask='-----     ', invert=False)
'-----fghij'
str2d.utils.chunk(inp: str, chunk_size: int) → map[source]¶

Split a long string into pieces of size=chunk_size

Parameters:
  • inp – Thing to be chunked
  • chunk_size – Size of each chunk
Returns:

map object that is an iterable of strings Attention: the last incomplete chunk is dropped (not included)

 
>>> tuple(chunk('abcdefghij', 2))
('ab', 'cd', 'ef', 'gh', 'ij')

>>> tuple(chunk('abcdefghij', 3))
('abc', 'def', 'ghi')

Randomly shuffle a string.

  • seed – random seed passed to random

str, a new string consisting of inp randomly shuffled
