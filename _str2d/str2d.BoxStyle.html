str2d.BoxStyle

class str2d.BoxStyle(value, names=<not given>, *values, module=None, qualname=None, type=None, start=1, boundary=None)[source]

Enumerate different box styles. They can be used to draw boxes around text. The box styles are made up of BoxParts objects that contain the box drawing characters. The main purpose of this class is to provide a way to apply box drawing over Str2D objects.

You can see all the box styles by calling the swatches method.

from str2d import BoxStyle

BoxStyle.swatches()
